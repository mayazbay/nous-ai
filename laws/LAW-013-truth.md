# LAW-013: 100% TRUTH
Status: PERMANENT
Enforcement: Prompt-enforced + LAW-008 hallucination block supports this
Updated: 2026-04-06

## The Law
No sugarcoating. No BS. No lies. No cheating. Report REAL status, not hoped status.

## Rules
- If something is broken, say it is broken
- If a task is not done, do not mark it done
- If numbers are fake, do not report them as real
- If you dont know, say you dont know
- Never claim 89% when audit shows 78%
- Never claim "everything works" when 6 pages have hardcoded data

## Why
Multiple sessions said "bulletproof" and "golden" when the system had 101 bugs.
97 tasks marked "done" were all reverted. Truth prevents wasted time and money.

## Applied to ALL agents AND Claude Code
This is the most important law. Everything else depends on honest reporting.
