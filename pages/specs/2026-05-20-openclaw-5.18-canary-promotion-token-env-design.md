## Plan (Codex impl steps)

### Token generation
```bash
# Run once on Air, capture the output (Madi sees this single value; nowhere else)
openssl rand -hex 32 > /tmp/oc-token.txt
echo "OPENCLAW_GATEWAY_TOKEN=$(cat /tmp/oc-token.txt)" >> ~/nous-agaas/.env
chmod 600 ~/nous-agaas/.env
rm /tmp/oc-token.txt
```

### Production plist edit

Air's `~/Library/LaunchAgents/com.nous.openclaw.plist` (or whatever Codex's existing prod plist is named):
```xml
<key>EnvironmentVariables</key>
<dict>
  ... existing keys ...
  <key>OPENCLAW_GATEWAY_TOKEN</key>
  <string>${OPENCLAW_GATEWAY_TOKEN}</string>
</dict>
```

Reload: `launchctl unload && launchctl load`.

### Swap container (the cutover moment)

```bash
ssh air '
  docker stop openclaw &&
  docker rm openclaw &&
  source ~/nous-agaas/.env &&
  docker run -d \
    --name openclaw \
    --restart unless-stopped \
    -p 127.0.0.1:18789:18789 \
    -e OPENCLAW_GATEWAY_TOKEN="$OPENCLAW_GATEWAY_TOKEN" \
    -v /Users/madia/nous-agaas/skills:/opt/nous-agaas/skills:ro \
    -v /Users/madia/nous-agaas/wiki:/opt/nous-agaas/wiki:ro \
    -v /Users/madia/nous-agaas/openclaw/home-node-openclaw-prod-20260520T1900Z:/home/node/.openclaw \
    ghcr.io/openclaw/openclaw:2026.5.18
'
```

### Verification (must all GREEN before tearing down canary)
