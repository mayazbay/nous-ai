# Cerebro VMS Requirements
Source: 3 spec docs from Satory team (Daniyar)

## Core Modules
1. Camera Management: add/edit/delete cameras, groups, health monitoring
2. Event Processing: ISAPI events → DB → UI feed
3. Violation Workflow: detect → review → approve → submit to ERAP
4. Reporting: daily/weekly/monthly, per-camera, per-operator
5. User Management: roles (admin, operator, viewer), audit trail
6. Map Integration: camera locations, traffic density overlay
