# LESSON-110 (alias) — Classify before requesting access

> **Migrated per RULE ZERO** — absorbed into [[agent-quality]] skill (AP-11).

**Original takeaway:** Don't ask "give me access to X" — first classify what
X is (project / person / system) and what kind of access (read / write /
admin) is actually needed, then ask precisely.
